Javascript learning
JavaScript, est un langage de programmation qui est l'une des technologies de base du World Wide Web, aux côtés de HTML et CSS. Plus de 97 % des sites Web utilisent JavaScript côté client pour le comportement des pages Web, incorporant souvent des bibliothèques tierces.
Java script is wrold number 1 most popular programming language in 2019.
Lorsque vous chargez une page sur Internet votre ordinateur a envoyé un dossier de votre ordinateur a envoyé un dossier de fichiers comprenant au moins un fichier html ce fichier html est capable de charger javascript avec quelque chose appelé (script tag) du même dossier ou ailleurs sur Internet.
Javascript est le seul côté client langage, ce qui signifie qu'il attend d'exécuter le code jusqu'à ce qu'il soit sur votre ordinateur à l'aide d'un programme appelé le navigateur, chaque navigateur a son propre moteur qui exécute réellement le code.
Client side codes :
Le code côté client est un code qui s'exécute dans votre navigateur. Il utilise toujours JavaScript, car c'est le seul langage que chaque navigateur comprend (pour l'instant). Lorsque les gens découvrent pour la première fois le code côté client.
Les moteurs JavaScript n'étaient à l'origine utilisés que dans les navigateurs Web, mais sont désormais des composants essentiels de certains serveurs et de diverses applications. Le système d'exécution le plus populaire pour cette utilisation est Node.js
